  7. Azure Firewall Premium (2)         - Centralized threat protection
  8. Microsoft Backbone                 - Hub-to-hub connectivity

CONNECTIVITY FLOWS (5 Core Patterns):
  1. VPN Tunnels: 10 branch offices → nearest regional hub
  2. VNet Links: 3 Azure VNets → hub routing (automatic)
  3. ExpressRoute: 2 data centers → hub gateways (BGP)
  4. Hub-to-Hub: US hub ↔ EU hub (automatic failover)
  5. Firewall: All traffic inspected (centralized enforcement)
